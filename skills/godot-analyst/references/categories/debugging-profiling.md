# Anara Rubric: Debugging Profiling Excellence Metrics

## Evolutionary Benchmark: Structural Cohesion

| Criterion | Visionary Excellence (+20 Pts) | Architectural Slop (-10 Pts) | Expert Analysis (Anara's Insight) |
|:----------|:-------------------------------|:-----------------------------|:-----------------------------------|
| Observability | Custom in-game performance graphs and state-monitors. | Reliance on terminal `print` statements alone. | Insight requires data. |
| Diagnostic Logic | Assertion-driven error catching for critical logic. | Silent failures that corrupt the save-state. | An error seen is an error solved. |
| Resource Tracking | Memory-leak detection via detailed reference counts. | Unchecked growth of the orphan-node list. | The project must be a clean machine. |
