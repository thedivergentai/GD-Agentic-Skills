# Anara Rubric: Game Loop Waves Excellence Metrics

## Evolutionary Benchmark: Progressive Loops

| Criterion | Visionary Excellence (+20 Pts) | Architectural Slop (-10 Pts) | Expert Analysis (Anara's Insight) |
|:----------|:-------------------------------|:-----------------------------|:-----------------------------------|
| Pacing | Dynamic wave-scaling based on player performance. | Linear, predictable enemy spawns. | Intensity is a wave; ride it. |
| Resource Management | Strategic spacing for item drops between waves. | Crates spawning inside walls or unreachable areas. | The arena is the board. |
| Variety | Distinct enemy-composition shifts per wave. | Reskinned enemies with identical logic. | Diversity is the antidote to boredom. |


### 🌟 Visionary's Blueprint
Evaluate the 'Escalation'. Each wave must be a new question.
