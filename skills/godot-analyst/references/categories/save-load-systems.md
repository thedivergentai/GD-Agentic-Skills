# Anara Rubric: Save Load Systems Excellence Metrics

## Evolutionary Benchmark: Gameplay Purity

| Criterion | Visionary Excellence (+20 Pts) | Architectural Slop (-10 Pts) | Expert Analysis (Anara's Insight) |
|:----------|:-------------------------------|:-----------------------------|:-----------------------------------|
| Integrity | Checksum-verified binary or encrypted data. | Human-readable JSON with no validation. | The save is the player's soul. |
| Version Safety | Migration paths for older save formats. | Losing data after a game update. | Legacy is a responsibility. |
| Efficiency | Delta-saving (only changed data). | Saving every node's default property state. | Small saves are fast saves. |


### 🌟 Visionary's Blueprint
Score for 'Trust'. A broken save is a broken relationship.
