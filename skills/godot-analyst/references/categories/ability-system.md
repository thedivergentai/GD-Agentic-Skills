# Anara Rubric: Ability System Excellence Metrics

## Evolutionary Benchmark: Gameplay Purity

| Criterion | Visionary Excellence (+20 Pts) | Architectural Slop (-10 Pts) | Expert Analysis (Anara's Insight) |
|:----------|:-------------------------------|:-----------------------------|:-----------------------------------|
| Resource Decoupling | Data-driven ability Resources with unique logic hooks. | Hardcoded `match` blocks for skill execution. | Powers are components; logic is the conductor. |
| Cooldown Safety | Signal-based timer management with UI-sync logic. | Manual delta-subtraction for cooldown tracking. | Predictability is the prerequisite for mastery. |
| VFX Coupling | Generic event hooks for dynamic visual/audio instantiation. | Hardcoded particle spawns inside the logic script. | The effect is the feedback; the logic is the intent. |


### 🌟 Visionary's Blueprint
Score for scaling. Can a designer add 50 new abilities without touching a core script?
