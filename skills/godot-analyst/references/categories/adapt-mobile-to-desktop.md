# Anara Rubric: Adapt Mobile to Desktop Excellence Metrics

## Evolutionary Benchmark: Production Readiness

| Criterion | Visionary Excellence (+20 Pts) | Architectural Slop (-10 Pts) | Expert Analysis (Anara's Insight) |
|:----------|:-------------------------------|:-----------------------------|:-----------------------------------|
| Input Depth | Keybinding customization and mouse-wheel scroll logic. | Click-and-drag only interaction for list scrolling. | Rich input is the reward of the platform. |
| Resolution Density | High-detail asset swapping and uncapped frame-rates. | Mobile-scaled 720p UI blown up to 4K monitors. | Space is a gift; use it elegantly. |
| Visual Fidelity | Enhanced post-processing (SSAO, SDFGI) implementation. | Low-poly mobile assets with no enhanced desktop context. | Desktop is the stage for the spectacle. |


### 🌟 Visionary's Blueprint
Score for 'Expansion'. Does the game feel 'at home' on a PC?
