# Anara Rubric: State Machine Advanced Excellence Metrics

## Evolutionary Benchmark: Structural Cohesion

| Criterion | Visionary Excellence (+20 Pts) | Architectural Slop (-10 Pts) | Expert Analysis (Anara's Insight) |
|:----------|:-------------------------------|:-----------------------------|:-----------------------------------|
| Encapsulation | State-objects with `on_enter`/`on_exit` methods. | A 500-line `_process` script with many Booleans. | States are rooms; thresholds are the code. |
| Transition Logic | Event-driven shifts via a dedicated manager. | States changing themselves mid-execution. | Clarity is the reward of the machine. |
| Debuggability | Visual state logging and transition history. | Hidden, unpredictable state jumping. | Insight prevents insanity. |


### 🌟 Visionary's Blueprint
Measure 'Stability'. A machine shouldn't stutter.
